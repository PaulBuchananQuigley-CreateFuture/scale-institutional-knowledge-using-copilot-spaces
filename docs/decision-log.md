# Decision Log

A lightweight index of all decisions made on OctoAcme projects. For each entry, link to the full [Decision Record](templates/decision-record.md).

---

## How to use this log

1. When a decision is made, create a new file in `docs/decisions/YYYY-MM-DD-short-title.md` using the [Decision Record template](templates/decision-record.md).
2. Add a row to this table.
3. Keep the status up to date — mark decisions as **Superseded** if they are later reversed or replaced.

---

## Decision Statuses

| Status | Meaning |
|---|---|
| **Proposed** | Under discussion; not yet finalised. |
| **Accepted** | Agreed and in effect. |
| **Superseded** | Replaced by a newer decision (link to replacement). |
| **Deprecated** | No longer relevant; context has changed. |
